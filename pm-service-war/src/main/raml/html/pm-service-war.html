get /subscription

Get filtered list of subscriptions

  • Request
  • Response

Query Parameters

  • Name: (string)

    Filter of subscription name. Multiple values can be provided.

    Example:

    Name=PM_02_328-13543222
  • Type: (string)

    Filter of subscription type. Multiple values can be provided.

    Example:

    Type=STATISTICAL
  • Status: (string)

    Filter of subscription status. Multiple values can be provided.

    Example:

    Status=INACTIVE&Status=ACTIVE
